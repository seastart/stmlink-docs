| 参数 | 默认 | 说明 | 备注 |
| --- | --- | --- | --- |
| stream_model | 3 | 流媒体模式[查看](https://www.yuque.com/anyconf/rtcengine/pfzomlthc9l03830#Cz51I) | 入会前设置 |

| stat_interval | 0 | 上下行回调时间间隔 | 会中可以随意设置 |
| speaker_interval | 0 | 音柱信息回调时间间隔 | 会中可以随意设置 |
| enable_stream_log | 0 | 使能流媒体日志 | 入会前设置 |
| enable_audio_record | 0 | 使能本地录音 | 入会前设置 |
| mcu_track | 1 | mcu轨道设置 | 会中可以随意设置 |
| den_model | 2 | 音频降噪设置<br/>0：声学，1：ai，2：ai+声学，3，关闭 | 会中可以随意设置 |

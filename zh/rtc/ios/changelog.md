### [1.0.0] - 2022.03.18
#### 新增
+ 新增终端类型`RTCTerminalType`；
+ 新增码流事件名称`RTCEngineKitEventNameStreamsEvent`；
+ 初始化配置参数`RTCEngineConfig`新增终端类型`terminalType`和终端描述`terminateDesc`属性；
+ 新增变更码流信息`changeWithStreams:changeType:`；
+ 新增类方法初始化RTC服务此方法外部可以持有和管理`RTCEngine`对象；
+ 初始化RTC服务新增`userModel`参数；
+ 新增只读属性`engineConfig`和`userModel`；

#### 变更
+ 原变更用户信息移除变更操作类型`changeType`参数；
+ 原初始化RTC服务方法`initializeWithConfig`新增`delegate`参数；
+ 原用户信息`RTCEngineUserModel`中`sdkno`属性变更为`linkid`，`sessionId`属性变更为`sessionKey`；
+ 房间信息`RTCEngineRoomMode` 对象中 ID 属性变更为`NSString`类型；
+ `RTCEngineConfig`对象移除`userModel`属性；
+ 原`sendMessageWithUserList:message:`方法变更为`sendMessageWithUserIds:message:`;

#### 优化
+ 内部逻辑优化。

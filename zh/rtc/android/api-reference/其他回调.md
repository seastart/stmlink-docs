## RTCResultListener

### 说明

`RTCResultListener` 为无返回数据的通用结果回调接口。

### 接口方法

#### onSuccess()
```java
void onSuccess();
```
方法说明：调用成功回调。  
参数说明：无。  
返回值说明：无（`void`）。

#### onFail(code)
```java
void onFail(int code);
```
方法说明：调用失败回调。  
参数说明：
- `code`：`int`，错误码。
返回值说明：无（`void`）。

## RTCResultListener2\<T\>

### 说明

`RTCResultListener2<T>` 为成功时携带结果对象的通用回调接口。

### 接口方法

#### onSuccess(t)
```kotlin
fun onSuccess(t: T)
```
方法说明：调用成功回调并返回结果对象。  
参数说明：
- `t`：`T`，成功结果对象。
返回值说明：无（`Unit`）。

#### onFail(code)
```kotlin
fun onFail(code: Int)
```
方法说明：调用失败回调。  
参数说明：
- `code`：`Int`，错误码。
返回值说明：无（`Unit`）。
